
# Little Lemon React Native App

![Little Lemon](./img/logo.png)

This repository contains the code for my beginner project, the **Little Lemon React Native App**, which I developed as part of my Coursera React Native Development course.

## About the App

**Little Lemon** is an imaginative restaurant-themed app that I conceptualized and brought to life. It has a variety of screens that demonstrate my evolving skills in React Native development.


## Getting Started

To explore the Little Lemon React Native App and dive into its codebase, follow these steps:

1. Clone this repository to your local machine using the following command:
```https://github.com/mfarrukh14/Little-Lemon-React-Native-App.git```

2. Navigate to the project directory:
```cd little-lemon-react-native```

3. Install the required dependencies by running:
```npm install```

4. Run the app on an Android/iOS emulator or a physical device:
```npx react-native run-android```
```npx react-native run-ios```

## Technologies Used

- React Native
- JavaScript
